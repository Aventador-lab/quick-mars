# quick-mars


## Usage

> locale

- clone repo

```bash
git clone https://github.com/Aventador-lab/quick-mars.git Qmars
cd Qmars
```



- install dependencies

```bash
yarn # npm install
```

- configuration locale env file

```bash
cp config/.env.locale.example config/.env.locale.js

# modified your mongodb config url
```

- start
```bash
yarn dev
```
